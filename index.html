# Crypto Portfolio Tracker This project is a simple cryptocurrency portfolio tracker to help you manage your investments. ## Features - View current prices of various cryptocurrencies. - Track your portfolio performance over time. - Get notifications for significant price changes. ## Installation 1. Clone the repository: ```bash git clone https://github.com/hwadley905-crypto/Prf-digital.git ``` 2. Navigate into the project directory: ```bash cd Prf-digital ``` 3. Install the required dependencies: ```bash npm install ``` ## Usage - To start the application, run: ```bash npm start ``` - Open your browser and navigate to `http://localhost:3000`. ## Contributing 1.
